# #1 Insertion Sort

[22,27,16,2,18,6]

1. Aşamalar

```
1. [22,27,16,2,18,6]
2. [16,22,27,2,18,6]
3. [2,16,22,27,18,6]
4. [2,16,18,22,27,6]
5. [2,6,16,18,22,27]
```

2. Big-O gösterimi

```
O(n^2)
```

3. Time Complexity

```
Average case
```

4. Dizi sıralandıktan sonra 18 sayısı hangi case kapsamına girer?

```
Average case çünkü 18 sayısı ortada
```

5. [7,3,5,8,2,9,4,15,6] dizisinin Insertion Sort'a göre ilk 4 adımını yazınız.

```

3. [2,3,5,7,8,9,4,15,6]
4. [2,3,4,5,7,8,9,15,6]
```

# #2 Merge Sort

[16,21,11,8,12,22]

1. Aşamalar

```
                        [16,21,11,8,12,22]

                [16,21,11]               [8,12,22]
              [16,21]   [11]          [8, 12]   [22]
            [16]   [21]    [11]    [8]    [12]    [22]

                        [8,11,12,16,21,22]
```

2. Big-O gösterimi

```
O(logn)
```
